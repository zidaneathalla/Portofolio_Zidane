# 🎨 Portfolio Website - React + Vite

Portfolio website Anda yang modern dan responsif dengan hero section animasi yang elegan, sempurna untuk keperluan magang atau mencari pekerjaan di bidang tech.

## ✨ Features

- ✅ **Hero Section** dengan typing animation
- ✅ **Project Showcase** dengan hover effects
- ✅ **Skills/Toolkit** dengan floating animation
- ✅ **Contact Form** dengan input validation
- ✅ **Fully Responsive** (Mobile, Tablet, Desktop)
- ✅ **Modern Gradient UI** dengan purple accent
- ✅ **Smooth Animations** dan Transitions
- ✅ **SEO Optimized**

## 🚀 Quick Start

### 1. Development Server
```bash
npm run dev
```
Buka: http://localhost:5173/

### 2. Build untuk Production
```bash
npm run build
```

### 3. Preview Build
```bash
npm run preview
```

## ⚡ 3 Langkah untuk Membuat Portfolio Anda

### 1. Siapkan Data Anda
- Nama dan bio
- Foto profile (JPG/PNG)
- 2-3 project terbaik (dengan link)
- List skills yang Anda kuasai
- Nomor HP dan email

### 2. Edit Files
**Paling penting:**
- `src/components/HeroSection.jsx` - Nama, bio, foto
- `src/components/Projects.jsx` - Daftar projects
- `src/components/Toolkit.jsx` - Skills Anda
- `src/components/Contact.jsx` - Nomor & email

**Opsional:**
- `src/styles/globals.css` - Ubah warna
- `src/assets/` - Tambah foto

### 3. Deploy
```bash
npm run build
# Pilih: Vercel / Netlify / GitHub Pages
# Ikuti panduan di DEPLOYMENT.md
```

## 💡 Tips

1. **Update Secara Regular**
   - Tambah project baru setiap bulan
   - Update skills yang sedang dipelajari
   - Refresh penampilan secara berkala

2. **Content is King**
   - Tulis deskripsi project yang clear
   - Tunjukkan real projects yang Anda buat
   - Highlight achievements/results

3. **Professional Touch**
   - Foto yang bagus dan profesional
   - Grammar dan spelling yang benar
   - Contact info yang valid dan aktif
