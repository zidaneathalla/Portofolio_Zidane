Penjelasan lengkap setiap file dan folder dalam portfolio project Anda.

## 📂 Folder Structure

```
porto/
├── src/
│   ├── components/          ← React Components
│   ├── styles/              ← CSS Files
│   ├── assets/              ← Images & Icons
│   ├── App.jsx              ← Main App Component
```

## 🗺️ Navigation Map

1. Hero Section text → Edit `src/components/HeroSection.jsx`
2. Projects → Edit `src/components/Projects.jsx`
3. Skills → Edit `src/components/Toolkit.jsx`
4. Contact → Edit `src/components/Contact.jsx`

1. Global colors/fonts → Edit `src/styles/globals.css`
2. Section specific → Edit corresponding CSS file

### Untuk Update Animations:
1. Global animations → `src/styles/globals.css` (dalam `@keyframes`)
2. Section animations → Corresponding CSS file

### Untuk Deploy:
1. Build → `npm run build`
2. Follow guide di `DEPLOYMENT.md`

## 💡 Pro Tips

1. **Backup sebelum besar-besaran edit**
   ```bash
   git add .
   git commit -m "Backup before major changes"
   ```

2. **Test di local dulu sebelum build**
   ```bash
   npm run dev
   ```

3. **Check terminal untuk error messages**
   Saat `npm run dev`, semua error akan tampil di terminal

4. **Gunakan browser DevTools untuk debug CSS**
   F12 → Inspect element → Debug styling

5. **Restart server kalau ada perubahan major**
   Ctrl+C (stop server) → `npm run dev` (start lagi)
